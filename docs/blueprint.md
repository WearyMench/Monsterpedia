# **App Name**: Monsterpedia

## Core Features:

- Monster List: Display a list of monsters, each with a name, photo, and short description.
- Monster Detail View: Display detailed information for a selected monster, including its full description and photo.
- Add Monster: Allow users to add new monsters with a name, photo, and description.
- Edit Monster: Allow users to edit existing monster information.
- Delete Monster: Enable users to delete monsters from the list.

## Style Guidelines:

- Primary color: Dark teal (#008080) for a mysterious and engaging feel.
- Secondary color: Light grey (#F0F0F0) for backgrounds and content separation.
- Accent: Deep purple (#800080) for interactive elements like buttons and links.
- Card-based layout for monster list items to display photo, name, and brief description.
- Use monster-themed icons for actions like edit and delete.
- Subtle transitions and hover effects on monster cards to enhance interactivity.
